# Repository Structure ``` 📁 sparklight-offer-manager ├── index.html # Main app file ├── README.md # Repo overview and setup instructions ├── .gitignore # Files to ignore in Git └── /assets # (optional) folder for proofs, CSS, JS ``` --- ## 📄 index.html This is the full working HTML file for the Offer Management System (already built). It contains: - Form for entering campaign data - Dropdowns for disclaimers and status - File upload field for proofs - Table view for all entries - CSV export functionality --- ## 🧾 README.md ```markdown # Sparklight Offer Management System A lightweight internal tool for tracking Sparklight promotional offers, disclaimers, and proof uploads. Built with HTML, CSS, and vanilla JavaScript. ## 🚀 Features - Add new offers with all key fields - Automatically track required disclaimers (Fiber, eero/Plume, Ookla) - Upload final creative proofs - Export all offers to CSV - Clean and responsive UI using vanilla HTML/CSS ## 🧠 How It Works Each new entry adds a row to the table. You can export all data to a CSV for record-keeping or review. ## 🧩 Setup 1. Clone the repository: ```bash git clone https://github.com/YOUR_USERNAME/sparklight-offer-manager.git cd sparklight-offer-manager ``` 2. Open `index.html` in your browser. 3.
